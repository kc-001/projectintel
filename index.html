What is a Project Managment Plan(PMP)?

A Project Managment Plan (or Work Plan) is a tool for outlining and managing a project's performance.

The Project Management Plan is the roadmap to your project's succes and should account for deliverables, schedule milestones, resources and costs.

How to develop a Project Managment Plan?

The Project Management Plan should be scaled to your project size. It can be as complex as need for large multi-year, multi-phased projects. Or is can be broad based but emcompassing of all project deliverables.
